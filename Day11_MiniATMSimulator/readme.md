This project is a Mini ATM Simulator built in the C programming language.  
It allows a user to perform basic banking operations such as:
- Check Balance  
- Deposit Money  
- Withdraw Money  
- Exit
- program will runs in a loop, allows up to 5 transactions per session, and ensures a minimum balance of ₹500 is always maintained.
