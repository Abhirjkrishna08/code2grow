A C program for college canteen billing system using loops and conditionals
which calculates bills for multiple customers, applies discounts, and shows total revenue
Concepts: Loops, Conditional Statements, Arithmetic Operations


## Sample Output
=========== Smart Canteen Billing System ===========
Customer 1 Bill:
Sandwich x 3 = ₹240
Pizza Slice x 2 = ₹300
Total: ₹540
Discount (10%): ₹54
Final Bill: ₹486

Customer 2 Bill:
Burger x 2 = ₹240
Coffee x 3 = ₹180
Juice x 1 = ₹50
Total: ₹470
Final Bill: ₹470

================== Canteen Summary ==================
Total Customers Served: 2
Total Revenue: ₹956
